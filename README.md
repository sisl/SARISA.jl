# SARISA

This Julia package is a wrapper for my modified APPL code that solves rho-POMDPs.

---------------

## Running the Code
-------------------
The code can be called as follows:
```
sarisa(pomdpx_file::String, timeout::Real=30, variant::String="MN"; filename="temp.policy")`
```
